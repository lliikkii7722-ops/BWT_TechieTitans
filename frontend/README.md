# Frontend – FocusFlow AI

This directory contains the VS Code Extension responsible for integrating FocusFlow AI directly into the developer’s workflow.

## Purpose

The frontend layer acts as the interaction bridge between the developer and the AI system. It captures real-time context and displays AI-generated insights inside the IDE.

## Core Responsibilities

- Monitor active files and editor activity  
- Capture Git metadata and repository context  
- Send contextual data to the backend AI engine  
- Display AI responses inside VS Code  
- Provide command palette integrations  

## Tech Stack

- TypeScript  
- VS Code Extension API  

## Planned Modules

- Context Listener  
- Command Controller  
- API Communication Layer  
- UI Panel / Webview Components  

This layer ensures seamless developer interaction without disrupting workflow.
